To Install Clarity desgin system:
1)install node js
2)install git bash
3)Goto the below link:
https://vmware.github.io/clarity/get-started

And follow the below 3 steps:

Clone the seed app:

git clone https://github.com/vmware/clarity-seed.git

Install the dependencies:

goto clarity-seed folder and then install npm as below:

npm install

Run the seed app:

npm start
--------------------------------------------------------------

Once you do this, then only UI changes will refect in browser

Type as below in the Web browser:
http://localhost:4200/
